## 基础知识杂记
### Go 规定：同一个文件夹里的 Go 文件，必须属于同一个 package

### Go 的 import 写的不是“包名”，而是模块路径 + 文件夹路径
你的 go.mod 里是：

```go
module go-demo
```

而你的工具文件在：
```go
go-demo/utils/util.go
```

所以在 index.go 里应该这样导入：
```go
import "go-demo/utils"
```

或者

```go
package main

import (
	"encoding/json"
	"fmt"
	"log"
	"net/http"
	"time"

	myutil "go-demo/utils"
)

```

### 在 Go 里，函数参数传递永远是值拷贝

### fmt包各方法快速总览

- Print 系：打印到终端

- Sprint 系：返回字符串

- Fprint 系：写到指定目标

- Scan 系：从终端读

- Sscan 系：从字符串读

- Fscan 系：从指定输入读

- Errorf：生成 error

- Append 系：追加到 []byte
